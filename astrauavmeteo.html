Політика конфіденційності
AstraUAVmeteo

Остання редакція: 26 квітня 2026 року

1. Статус проєкту

AstraUAVmeteo — це спеціалізований, суто некомерційний інструмент, розроблений для інструкторів та пілотів БПЛА літакового типу (фіксоване крило). Додаток не призначений для комерційного використання, не містить реклами чи прихованих платежів.

2. Нульовий збір даних (Zero-Data Policy)

Ми сповідуємо принципи абсолютної приватності. Додаток не збирає, не зберігає, не аналізує та не передає жодних персональних даних користувача на сервери розробника. Уся телеметрія залишається виключно у ваших руках.

3. Геопозиція та локальна обробка

Доступ до геолокації використовується виключно в момент запиту для формування актуального метеорологічного зведення, критичного для планування польоту. Ці дані обробляються суворо локально на вашому пристрої. Студія AstraCoreNova не має доступу до ваших координат або маршрутів.

4. Взаємодія зі сторонніми API

Додаток звертається до зовнішніх метеорологічних сервісів виключно з метою отримання технічних даних про погодні умови. Під час цих запитів не передаються жодні ідентифікатори пристрою, акаунти чи особиста інформація.

5. Зв'язок

Ми автономні, але відкриті до технічного діалогу. Запити та пропозиції приймаються за адресою: support@astracorenova.com.
